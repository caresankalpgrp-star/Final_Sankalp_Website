🚀 How to Upload to GitHub + Deploy

1
Extract ZIP — You'll get a folder called sankalp-interior/
2
Create GitHub Repo — github.com → New Repository → name it sankalp-interior → Private
3
Upload Files — In GitHub repo → "uploading an existing file" → drag the extracted folder contents
4
Deploy on Vercel — vercel.com → New Project → Import GitHub repo → Add environment variables
5
Add Env Variables in Vercel → Settings → Environment Variables (see README.md for the list)
⚠️ Important: The ZIP does NOT contain .env (your secret keys). After uploading to GitHub, add your Supabase keys directly in Vercel → Settings → Environment Variables. Never commit real keys to GitHub.
